Fawful

Fawful is a Discord Bot designed and developed by me, to replace as many other Bots as possible so the community would only have to use one.

The Fawful Bot has been in developement for about half a year now, with updates happening daily. If you are interested in using the Fawful Bot for your own personal server, feel free to invite the Beta Bot here! The main Fawful Bot is not available to any users until the developement is completely finished. If you'd still like to test it out, it is available in my own personal discord server.
