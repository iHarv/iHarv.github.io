iHarv's Hack

iHarv's Hack is a mod created for NSMBU from Nintendo. It is made using the Miyamoto Level Editor, and the mod will release early 2021.

This hack will include:

- 8 Levels

- 2 Boss Battles

- Themes inspired by the Newer Team

- Assets created by Rimea, Sierra and WillyMaker

- Music composed and remixed by Cyratu, Buwomp and iHarv

- Translations for Spanish, German and French Users

iHarv's Tileset Collision Template

From a Request by "in English, my name means Noble" and "Abyzzol", I decided to create an Extention to Rimea's Tileset Template. My Template fills in all the Collisions for the Tiles, so you don't have to. Of course if you'd like to, you can edit these to your own needs!

Step 1) Open up Miyamoto Level Editor and click on the Tilesets tab

Step 2) Click file > Import Tileset from file > iHarv's Tileset Collision Template

Step 3) Click file > Import Image > Rimea's Tileset Template, then Save and Quit

Then you're all done!

Special Thanks to Rimea to creating the standard Tileset Template, which has helped a lot of people make their own Custom Tilesets!
