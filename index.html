Fawful

Fawful is a Discord Bot designed and developed by me, to replace as many other Bots as possible so the community would only have to use one.

The Fawful Bot has been in developement for about half a year now, with developements happening daily. If you are interested in using the Fawful Bot for your own personal server, feel free to invite the Beta Bot here! The main Fawful Bot is not avalible to any users until the developement is completely finished. If you'd still like to test it out, it is avalible in my own personal discord server.

iHarv's Hack

iHarv's Hack is a mod created for NSMBU from Nintendo. It is made using Miyamoto Level editor, and the mod will release early 2021.

This hack will include:

- 8 Levels

- 2 Boss Battles

- Themes inspired by the Newer Team

- Assets created by Rimea, Sierra and WillyMaker

- Music composed and remixed by Cyratu, Buwomp and iHarv

- Translations for Spanish and French users
